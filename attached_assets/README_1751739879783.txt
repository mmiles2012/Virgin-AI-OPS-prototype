# METAR ML Extension for Replit

This module allows you to:

1. Parse METAR logs (one line per report, plain text format)
2. Detect significant weather events (TS, SN, FG, FZ)
3. Merge these features into your ML-ready delay dataset

## Usage

- Place METAR logs into a local folder, e.g. /data/metar/
- Use `parse_metar_file()` to convert the file into a dataframe
- Use `enrich_with_metar(delay_df, metar_df)` to add features

Example:

```python
from metar_enrichment import parse_metar_file, enrich_with_metar

metar_df = parse_metar_file("data/metar/KJFK_METAR.txt")
enriched = enrich_with_metar(delay_df, metar_df)
```
